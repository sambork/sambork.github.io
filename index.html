THE EASY WAY

WordPress.com

What is WordPress.com?

  • free Content Management System (CMS)* hosted online
  • blog-ready system
  • multi-language admin system

*CMS is a tool to manage websites with databases (for user accounts, generating dynamic content results based on search, etc.).

Example: Wordpress template (theme) for a blog website with sample articles.

PROS:

  • free website hosted online for you
  • easy to set up
  • no coding skills needed
  • ssl security out of the box (https)
  • free, responsive, and accessability-ready layouts
  • community oriented
  • offers tutorials and guides
WordPress.com community features

Easily discover, visit new blogs, and comment on posts.

CONS:

  • link: your-website-name.WordPress.com
  • freemium model (pay or code) impacting growth
  • limited possibilities (including SEO)

THE NORMAL WAY

Self-hosted CMS (WordPress.org)

What it means self-hosted CMS?

  • buying server space
  • uploading and setting up the CMS on your own
  • managing the CMS

PROS:

  • you have control over your website
  • you can use plugins to add features and functionalities (f.eg. SEO optimization)
  • you can upload all those fancy themes that aren't in the online repository
  • you can use Google/Adobe Analytics to measure activity on your website
  • easy to go from blog to portal but online store is also possible
  • you learn a popular CMS

WordPress.org admin dashboard overview

CONS:

  • you're responsible for website administration and security
  • you need to pay for server and domain

Some alternatives

Moodle Drupal Joomla

Tip:

choose a hosting service with Cpanel (below) which usually has automatic installers of popular CMS.

THE HARD WAY

DIY website

What it means DIY website?

  • all coding on your side
  • most likely you need to self host your website
  • best suited to business card-like websites

PROS:

  • you have a total control
  • makes it easy to create a low-resource website
  • ideal for a business-like website (no database needed)

CONS:

  • need to learn at least HTML5 & CSS
  • need to remember about responsive and accessible design on your own
  • might be time consuming to manage or update

Example website: International Women's Day by Apple.

TIP:

You can also use HTML5 free templates and edit them to your liking. There are many websites similar to HTML5 UP.
